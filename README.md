# ☁️ Computação em Nuvem — AWS Foundations

## Sobre

Este repositório reúne as atividades, anotações e práticas desenvolvidas ao longo da disciplina de **Computação em Nuvem**, baseada no curso **AWS Academy Cloud Foundations** da Amazon Web Services.

O objetivo é consolidar o aprendizado dos principais conceitos de cloud computing, com foco na utilização prática dos serviços da AWS.

---

## Conteúdos Abordados

* Introdução à Computação em Nuvem
* Modelos de serviço (IaaS, PaaS, SaaS)
* Infraestrutura global da AWS
* Serviços principais:

  * EC2 (máquinas virtuais)
  * S3 (armazenamento)
  * VPC (rede)
* Segurança na nuvem (IAM, Security Groups)
* Monitoramento e custos
